# Linux
整理自用
### 更改权限
```
chmod [user[opations]]
r 4
w 2
x 1
4+2+1 = 7 即最高权限 r + w + x
user: owner/group/others
最高权限 chmod 777 即所有用户都可读写运行
        chmod 770 即除ohters 外每个用户都可读写运行
```
### 文件相关操作
```
创建文件夹          mkdir + filename
设置权限            chmod + [n,n,n] + folderName
进入文件夹内部       cd + folderName
创建文件             touch + fileName
显示全部文件          ls -al

切换用户            su - (userName)

```

+ 编辑模式进入:i I o O a A r R

+ 按键说明:
```
          h ⬅️ 光标向左移动一个字符
          j ⬇️ 光标向下移动一个字符
          k ⬆️ 光标向上移动一个字符
          l ➡️ 光标向右移动一个字符
          ctrl + f 下翻一页
          ctrl + b 上翻一页
          ctrl + d 下翻半页
          ctrl + u 上翻半页
          + 光标移动到非空格符的下一行
          - 光标移动到非空格符的上一行
          0 移动到该行最前字符处
          $ 移动到该行最后字符处
          H 移动到这个屏幕的最上方一行的首字符处
          M 移动到这个屏幕的中央那一行的首字符处
          L 移动到这个屏幕的最下方一行的首字符处
          G 移动到这个文件的最后一行
          [n]G 参数n为数字 移动到这个文件的第 n 行
          gg 移动到这个文件的第一行
          [ n ] + Enter 光标向下移动 n 行
          ／[ key ] 向下寻找一个名称为 [ 参数key ] 的字符串
          ? [ key ] 向上寻找一个名称为字符串[ 参数key ]的字符串
          :[n1],[n2]s/[word1]/[word2]/g 在第n1行与n2行之间寻找word1这个字符串，并将其替换成word2
          :1,$s/word1/word2/g 从第一行到最后一行查找word1字符串，并将其替换成word2
          :1,$s/word1/word2/gc 从第一行到最后一行查找word1字符串，并将其替换成word2，在替换前显示提示信息给用户是否要替换
```
+ 删除、复制、黏贴
```
          x,X 在一行中 x 表示向后删除一个字符相当于 del 按键，X 为向前删除一个字符相当于Backspace
          [ n ]x n为数字，连续向后删除 n 个字符
          dd 删除光标所在的一整行
          [ n ]dd n为数字，删除光标所在向下n行
          d1G 删除光标所在到第一行的所有数据
          dG 删除从光标所在到最后一行的所有数据
          d$ 删除从光标所在处到该行的最后一个字符
          d0 删除从光标所在处到该行的最前面一个字符
          yy 复制光标所在的那一行
          [ n ]yy n为数字。复制光标所在的向下n行
          y1G 复制光标所在到第一行的所有数据
          yG 复制从光标所在到最后一行的所有数据
          y0 复制从光标所在处到该行的最前面一个字符
          y$ 复制从光标所在处到该行的最后一个字符
          p,P p为将已复制的数据在光标下一行黏贴，P则为黏贴在光标上一行。
```
